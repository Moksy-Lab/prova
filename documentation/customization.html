Customization

This section provides detailed instructions on how to customize various aspects of the Moksy e-learning templates. Whether you want to change images, text, styles, or add custom CSS, you'll find all the necessary information here.

Changing Images

To customize the images in the template, follow these steps:

  1. Locate the images you want to change in the assets/images directory. The images are organized into subdirectories like branding and general.
  2. Replace the existing images with your own images, making sure to keep the same file names to avoid broken links.
  3. If you want to use new file names, update the image source paths in the HTML files where the images are used. For example: 
               
          <img src="assets/images/branding/new-image.jpg" alt="New Image">

Changing Text

To update the text content in the template, follow these steps:

  1. Open the HTML files in your code editor. The main content is usually found in index.html and other section-specific HTML files.
  2. Locate the text you want to change and edit it directly within the HTML file. For example: 
               
          <p>Welcome to the Moksy e-learning template!</p>
  3. Save the changes and refresh your browser to see the updated content.

Changing Styles (CSS)

To customize the styles of the template, follow these steps:

  1. Open the main CSS file located at assets/styles/styles.css in your code editor.
  2. Locate the styles you want to change. You can use the browser's developer tools to inspect elements and find the relevant CSS rules.
  3. Edit the CSS rules to customize the appearance of the elements. For example: 
              
          .header {
            background-color: #ff0000;
            color: #ffffff;
          }
  4. Save the changes and refresh your browser to see the updated styles.

Customizing Components

The template includes various pre-designed components that you can customize to fit your needs. Follow these steps to customize components:

  1. Open the HTML file where the component is used. For example, index.html.
  2. Locate the component's HTML markup. For example, a button component might look like this: 
               
          <button class="btn btn-primary">Click Me</button>
  3. Edit the HTML markup and add or remove classes to customize the component. You can also add inline styles if necessary. For example: 
               
          <button class="btn btn-primary" style="background-color: #ff0000;">Click Me</button>

Adding Custom CSS

If you need to add custom CSS to the template, follow these steps:

  1. Open the main CSS file located at assets/styles/styles.css in your code editor.
  2. Add your custom CSS rules at the end of the file. For example: 
              
          .custom-banner {
            background-color: #00ff00;
            font-size: 2rem;
          }
  3. Save the changes and refresh your browser to see the custom styles applied to the elements.
